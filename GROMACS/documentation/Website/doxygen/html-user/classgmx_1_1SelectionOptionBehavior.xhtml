Description

Options behavior to allow using SelectionOptions.

This behavior wraps SelectionOptionManager, as well as all calls to SelectionCollection up to and including selection compilation.

To use selections through SelectionOption options in a ICommandLineOptionsModule, you need to

  • create a SelectionCollection object,
  • implement ITopologyProvider to return the topology or number of atoms to be used with the selections,
  • create and add a SelectionOptionBehavior, and call initOptions() to add common options for selection control,
  • use SelectionOption options to specify the selections used, and
  • evaluate the selections when running the module for the desired sets of coordinates (see SelectionCollection::evaluate()).

The SelectionOptionBehavior provides the following functionalities to the module:

  • Creates SelectionOptionManager and manages all calls to it.
  • Creates an option to provide an ndx file, and loads it when necessary.
  • Creates options to control general aspects of selections (see SelectionCollection::initOptions()), as well as a generic option to provide selections from a file.
  • After all options have been processed, provides an interactive command-line prompt for any missing selections.
  • Compiles the selections.

The behavior needs to be added before any options are created.
